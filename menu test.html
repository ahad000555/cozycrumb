About This System

What is This System?

This is a restaurant allergen checking system designed for kitchen staff to quickly identify which menu items contain specific allergens.

It helps prevent allergic reactions by providing instant access to allergen information for all menu items.

How to Use

  • By Allergen Tab: Click any allergen button to see all foods containing that allergen
  • By Food Tab: Select a category (Starters, Main, etc.) then click a food to see its allergens
  • Clear All: Removes all selections
  • Print: Prints the current results in a clean format

Important Notes

  • Always double-check with kitchen staff about cross-contamination risks
  • Menu items and ingredients may change - verify with chef
  • This system is a reference tool, not a replacement for proper communication
  • For severe allergies, inform the manager immediately
